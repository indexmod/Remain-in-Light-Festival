# Remain in Light Festival Web Page

## English

This is a single-page static website for the **Remain in Light Festival**, a Whitstable Carnival fundraiser.

### Features

- **Animated pastel sunset gradient** as the full-page background.
- **Large, magazine-style typography** for the festival title and subtitle.
- **Central Play button** for streaming live audio.
- **Artist list**, headliner, and extras clearly displayed.
- **Ticket link** and location information.
- **Fully responsive** layout for mobile and desktop.

### How it works

- The **Play button** (`#playBtn`) starts and pauses the live audio stream.  
- The background gradient animates smoothly using CSS `@keyframes`.  
- All content is in a **transparent central block**, letting the gradient fill the full page.

### Live Stream

Audio stream URL: `https://stream-mixtape-geo.ntslive.net/mixtape6`  
Clicking the play button starts the stream.

### Usage

Simply open `index.html` in a browser. The page is static and does not require a server.

## Русский

Это одностраничный статический сайт для **фестиваля Remain in Light**, организованного в рамках благотворительного проекта Whitstable Carnival.

### Особенности

- **Анимированный пастельный градиент заката** на весь экран.
- **Крупная типографика в стиле журналов** для заголовка и подзаголовка.
- **Центральная кнопка Play** для запуска аудио стрима.
- **Список артистов**, хэдлайнер и дополнительные активности.
- **Ссылка на билеты** и информация о локации.
- **Адаптивный дизайн**, поддержка мобильных и десктопных экранов.

### Принцип работы

- Кнопка **Play** (`#playBtn`) запускает и ставит на паузу аудиопоток.
- Анимация градиента реализована с помощью CSS `@keyframes`.
- Контент расположен в **прозрачном центральном блоке**, градиент виден на всей странице.

### Аудио стрим

URL потока: `https://stream-mixtape-geo.ntslive.net/mixtape6`  
Нажатие на кнопку Play запускает воспроизведение.

### Использование

Просто откройте файл `index.html` в браузере. Сайт полностью статический, сервер не требуется.
